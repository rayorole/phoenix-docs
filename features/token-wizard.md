# Token wizard

The token wizard is an exclusive feature built for Phoenix

### What is this?

The token wizard, allows you to launch your created tokens in seconds.

We support 3 launch types at the moment:

* 0 Bundle block launch
* Delayed bundle
* Bundle on demand
